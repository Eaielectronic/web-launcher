LE LAUNCHER OFFICIEL

Rejoignez l'aventure instantanément. Téléchargez le launcher, connectez-vous et plongez au cœur du serveur. Tout est configuré pour vous.

Version macOS

Support natif Apple Silicon & Intel (macOS 10.13+)

Télécharger .DMG

Note de sécurité macOS :

Si vous voyez "Application endommagée" ou "Développeur non identifié" :

  1. Ouvrez votre Terminal (Applications > Utilitaires)
  2. Collez la commande (utilisez sudo si l'app est dans Applications) :
    sudo xattr -cr /Applications/LauncherCreat.app
  3. Ou faites Clic-droit > Ouvrir dans le Finder.

Version Windows

Compatible Windows 10 & 11 (Architecture x64)

Télécharger .EXE

* Si l'écran SmartScreen s'affiche, cliquez sur "Informations complémentaires" puis sur "Exécuter quand même".

Version Linux

Format universel AppImage (Architecture x64)

Télécharger .AppImage

Instructions :

  1. Après le téléchargement, rendez le fichier exécutable.
  2. chmod +x "LauncherCreat-1.0.0.AppImage"
  3. Double-cliquez pour lancer le jeu.
